How to compile gapc on Windows 10:

1) Install Linux Sub-System
	1.1) go Start -> Settings -> For Developers : Click Developer-Mode
	1.2) Open Windows Power Shell then type: Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux
	1.3) restart computer
	1.4) Download Ubuntu app from Windows-Strore
	1.5) Initialize installation by setting Username and Password
2) Prepare compiling and installation of gapc by installing requiered sources: 
	2.1) Open Ubuntu app
	2.2) install compiler: sudo apt update
			       sudo apt install gcc
			       sudo apt install g++ (for some reason u need both)	
	2.3) install flex    : sudo apt-get install flex
	2.4) install bison   : DO NOT :sudo apt-get install bison !! 
			       INSTEAD go on the bison website and istall the latest version
	2.5) install boost   : sudo apt-get install libboost-all-dev (takes some time)
	2.6) install GSL     : sudo apt-get install libgsl-dev
	
3) compile gapc: 
	3.1) ./configure 
	3.2) make 
	3.3) (sudo) make install
